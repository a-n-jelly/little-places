# Little Places design — external references

Use these as **inspiration or process hints**, not as a second design system. Implementation must go through `src/styles/theme.css`.

| Resource | Use for |
| -------- | ------- |
| [Owl-Listener/designer-skills](https://github.com/Owl-Listener/designer-skills) | Process ideas: research, design-system audit mindset, optional Claude Code plugins/commands |
| [bergside/awesome-design-skills](https://github.com/bergside/awesome-design-skills) | Mood / palette / typography **direction** — preview via [typeui.sh design skills](https://typeui.sh/design-skills); map any colour choices into `theme.css` |
| [nextlevelbuilder/ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill) | Broad UI/UX checklist — cherry-pick ideas only |

**Pull commands** (optional): `npx typeui.sh list` / `npx typeui.sh pull <slug> --dry-run` to inspect presets before adopting any language into this project’s tokens.
